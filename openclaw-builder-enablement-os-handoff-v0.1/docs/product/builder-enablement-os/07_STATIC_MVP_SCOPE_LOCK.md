# Static MVP Scope Lock

## Allowed now

- Static mock SpecGraph fixtures.
- Static Workbench copy refinement.
- Static demo scenarios.
- Static code-object explorer mock fixture.
- Documentation and product locks.
- QA checklist and Spec CI mock checks.
- Codex handoff prompts.
- Evidence and artifact manifests.

## Requires Point lock

- New dependencies.
- Backend.
- API.
- Database.
- Authentication.
- Deployment.
- Runtime mutation.
- Persistent storage.
- Real AI execution.
- External connectors.
- MCP.
- Local parser process.
- LSP bridge.
- Tree-sitter or parser dependency.
- SCIP, Kythe, CodeQL, or Joern.
- Real repo ingestion.
- Source-code upload.
- Commits, pushes, PRs, or publication.

## Stop conditions

Stop and ask Point if scope requires new dependency, runtime execution, backend/API, real repo ingestion, source code, secrets, workflow automation semantics, or product conflict with SpecGraph lock.
