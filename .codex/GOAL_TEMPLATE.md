# Goal Template

Use this template to define tasks for the Codex agent. Each goal should include:

* **Title:** A concise description of the objective.  
* **Objective:** Detailed description of what the agent must achieve.  
* **Autonomy level:** The allowed scope for the agent (e.g. A1, A2, A3).  
* **Allowed paths:** Which folders/files the agent may modify.  
* **Forbidden actions:** What the agent must not do (e.g. add dependencies, modify protected surfaces).  
* **Input:** Any input files or context required.  
* **Required work:** The steps the agent should perform.  
* **Acceptance criteria:** Conditions for success.  
* **Validation commands:** Commands to verify the work.  
* **Stop conditions:** When to stop and ask for help.  
* **Final response format:** Structured output the agent must return.
