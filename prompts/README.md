# Prompts

This folder contains reusable prompt templates and handoff instructions for ChatGPT and Codex agents. Each prompt should be clearly documented and versioned if it evolves over time.

Example files:

- `CODEX_GOAL_TEMPLATE.md` – a template for writing new Codex goals.  
- `RESEARCH_TO_BUILD_PROMPT.md` – converts research outputs into build tasks.  
- `REVIEW_AGENT_PROMPT.md` – instructs the Review agent on how to review tasks.  
- `QA_PROMPT.md` – prompts for QA evaluations.  
- `AGENT_HANDOFF_TEMPLATE.md` – general format for handing off work between agents.
