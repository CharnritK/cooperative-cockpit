# Prompts

This folder contains reusable prompt templates and handoff instructions for ChatGPT and Codex agents. Each prompt should be clearly documented and versioned if it evolves over time.

Current prompt files:

- `CODEX_STATIC_MVP_SCOPE_GUARDRAIL_PROMPT.md` – required guardrail prompt for Codex tasks touching the static MVP.

Add new prompt templates only when they have a concrete user or agent workflow. Do not list placeholders that are not present in this folder.
