# Roadmap

This document provides a high‑level roadmap for the OpenClaw Cooperative Cockpit project. Major milestones include:

1. **Repository setup** – Establish agent workflows, approval and concurrency policies, and validation scripts. *(Current phase)*
2. **Static MVP assimilation** – Bring the static HTML/CSS/JS prototype into the repo under `apps/static-mvp/` and record the handoff.  
3. **QA hardening** – Improve QA coverage, refine documentation, and ensure that the static MVP meets acceptance criteria.  
4. **Feature iteration** – Plan and implement next features, subject to Point approval.
