Cooperative Cockpit Design System

A refined, faithful system extracted from the static MVP. It tunes hierarchy, density, and state clarity while preserving the product identity — a quiet, inspectable, auditable workflow studio for turning fuzzy work into a governed builder handoff.

01 · Brand
Mark, voice & shell

The shield mark signals governance and safety. The voice is operational and precise — short, factual labels; never marketing. Imagery is sparse; the interface is the product.

08 · Responsive & accessibility
Density down, never meaning
Responsive
  • ≥ 1180px: full three-region shell, optional 340px right inspector.
  • 760–1180px: inspector collapses to an overlay; governance strip wraps to 2 columns.
  • ≤ 760px: left rail collapses to a 72px icon rail; labels and the readiness meter hide.
  • Canvas screens pan/zoom; data screens reflow panels to a single column.
Accessibility
  • Focus ring: 2px accent at 88%, 2px offset, on every interactive element.
  • State never relies on hue alone — chips pair color with a text label.
  • Body text ≥ 14px; muted text holds AA contrast on white surfaces.
  • Honors prefers-reduced-motion: pulses and transitions collapse.
  • Skip-link to main; visible keyboard focus on nav items.
