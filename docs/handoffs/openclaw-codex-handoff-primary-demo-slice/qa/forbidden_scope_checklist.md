# Forbidden Scope Checklist

Codex must not:

- Add backend code.
- Add API calls.
- Add auth.
- Add database/storage.
- Add deployment scripts.
- Add runtime workflow execution.
- Add real agent orchestration.
- Add external connectors.
- Implement MCP.
- Add dependencies.
- Add secrets.
- Touch files outside allowed paths.
- Add a ninth page.
- Rename existing pages without Point lock.
- Use user-facing labels that imply live execution.
- Produce real exports or repo writes.
- Claim validation passed without running commands.
