# Stop Conditions

Stop if:

- Dependency required.
- Backend/API/auth/database/deployment/runtime work required.
- Real agent orchestration required.
- Real connector or MCP work required.
- Files outside allowed paths must change.
- Product or architecture decision required.
- Page count must expand.
- Validation fails and cannot be fixed within scope.
- Secrets or credential-like material are found.
- UI implies real execution or repository write.
