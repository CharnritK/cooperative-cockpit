# Demo Scenarios

## Scenario A — Personal portfolio website

Low complexity. Use Guided Flow.

The system turns vague intent into audience, positioning, proof points, page structure, visual direction, CTA, static page architecture, QA checklist, and Portfolio Website Build Packet.

Reusable pattern: Personal Portfolio Website Blueprint.

## Scenario B — Power BI + Databricks dashboard

Medium complexity. Use Lineage / Impact Map.

The system turns dashboard intent into business question, KPI contract, Databricks source, transformation logic, semantic model, DAX/report behavior, reconciliation QA, and BI Dashboard Change Packet.

Reusable pattern: Power BI + Databricks Dashboard Blueprint.

## Scenario C — OpenClaw personal AI config

High complexity. Use Control Plane.

The system turns config intent into Telegram channel, gateway, mission cockpit, skill registry, permission policy, secrets boundary, logs/evidence, rollback, and Personal AI Config Change Packet.

Reusable pattern: Personal AI Control Plane Blueprint.

## Scenario D — Code-object explorer lens

Technical specialist scenario. Use Code Review Lens.

The system shows selected code object, bounded relationships, exact evidence, findings, annotations, and Technical Code Review Packet.

Reusable pattern: Evidence-linked Code Review Blueprint.
