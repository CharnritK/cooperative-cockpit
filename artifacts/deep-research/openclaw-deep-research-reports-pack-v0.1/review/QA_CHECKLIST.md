# QA Checklist — Deep Research Reports Pack

- [ ] Package has README and file index.
- [ ] Package has manifest.
- [ ] Package has at least one report file.
- [ ] Package has claim/source/gap registers.
- [ ] Package has Codex persistence goal.
- [ ] Package states static MVP constraints.
- [ ] Package does not claim app implementation.
- [ ] Package does not claim repo mutation.
- [ ] Package does not include secrets.
- [ ] Package includes Point review gate.
- [ ] Package includes missing-input caveats.
- [ ] Package can be persisted without touching app source files.
- [ ] Codex goal includes allowed paths, forbidden actions, required work, acceptance criteria, validation commands, stop conditions, and final response format.
